# 🐾 Woofstagram

Um Instagram para pets feito com React Native. Projeto desenvolvido para a disciplina de Interface Gráfica (UEPB).

---

## Telas

- **SignIn** — tela de boas-vindas com acesso ao cadastro
- **Cadastro** — formulário completo com dados do usuário e do pet
- **Feed** — tela inicial com pets em alta e posts recentes

---

## Como rodar

**Pelo Expo Snack (sem instalar nada):**

Acesse [snack.expo.dev](https://snack.expo.dev) e importe os arquivos do projeto.

---

## Tecnologias

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [React Navigation](https://reactnavigation.org/) — Stack + Bottom Tabs
- [Formik](https://formik.org/) — gerenciamento de formulário
